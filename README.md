# Cuestionario Pruebas

Proyecto web estático compuesto por `index.html`, `app.js` y `styles.css`.

## Ejecutar en local

```bash
npm install
npm start
```

Luego abrí:

```bash
http://localhost:3000/index.html
```

## Estructura

- `index.html`: página principal
- `app.js`: lógica del cuestionario
- `styles.css`: estilos de la interfaz
